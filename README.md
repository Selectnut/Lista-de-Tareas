## Getting Started

To get a local copy up and running follow these simple example steps.

--Node

--Composer

### Install

1. Clone the repo
   ```sh
   git clone git@github.com:gioudi/Lista de Tareas.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

   ```sh
   yarn install
   ```
